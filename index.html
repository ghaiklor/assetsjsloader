AssetsJSLoader

JS-loader for assets which loads CSS and JS files asynchronously

View project on GitHub

Welcome to AssetsJSLoader

AssetsJSLoader help you don't worry about loading CSS and JS files with non-blocking and lazy loading...

Just call AJL constructor with provided parameters: 

    
        (function() {
            AJL.Environment('debug', ['foo.js', 'bar.js', 'style.css']);
        })();
